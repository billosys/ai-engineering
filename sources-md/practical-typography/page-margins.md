page marginsOne inch is not enough

Page margins set the default territory your text occupies on the page. Because they determine the maximum width of the text block, page margins have the greatest effect on <a href="line-length.html" class="xref">line length</a>. (<a href="point-size.html" class="xref">Point size</a> also affects line length, though more finely.) As page margins increase, line length decreases, and vice <span class="no-hyphens">versa.</span>

A gutter margin is extra space on the inner edge of a page, used to account for a binding. In a duplex (two-sided) document, the gutter margin will automatically alternate <span class="no-hyphens">sides.</span>

<div id="in-print" class="subhead">

[in print](#in-print)

</div>

Most word processors default to page margins of one inch. On standard 8.5″ × 11″ paper, that produces a line length of 6.5″. That was fine for the <a href="monospaced-fonts.html" class="xref">monospaced fonts</a> of the typewriter era, which used a lot of horizontal space. But for proportional fonts, they’re too <span class="no-hyphens">small.</span>

At 12 point, left and right page margins of 1.5–2.0″ will usually give you a comfortable line length. But don’t take that range as an absolute—focus on getting the number of characters per line into the right range (see <a href="line-length.html" class="xref">line length</a>). The smaller the point size, the larger the page margins will need to be, and vice <span class="no-hyphens">versa.</span>

<div id="on-the-web" class="subhead">

[on the web](#on-the-web)

</div>

Originally, web browsers had no concept of margins—text simply flowed from one edge of the window to the other. That wasn’t a huge problem at the time, because computer screens were much smaller. With the advent of CSS, web designers could use the `margin` and `padding` properties to create space between text and the edge of its containing <span class="no-hyphens">block.</span>

Nevertheless, the web has never shaken the edge-to-edge design idiom. Many of today’s <a href="websites.html" class="xref">websites</a> still look like they were created by designers who got paid by the square inch. Just as it’s important in print to let go of <a href="typewriter-habits.html" class="xref">typewriter habits</a>, it’s important on the web to let go of the 1994 <span class="no-hyphens">habits.</span>

To preserve text legibility, web pages need big margins too. There isn’t one margin size that will work for all web pages, but the core advice is the same as on the printed page—focus on <a href="line-length.html" class="xref">line length</a>. As you do that, you’ll find your margins getting bigger. Don’t <span class="no-hyphens">panic.</span>

<div id="fear-of-white-space" class="subhead">

[Fear of white space](#fear-of-white-space)

</div>

“But if I use bigger margins, won’t a lot of the page be empty?” Sure. Is that a <span class="no-hyphens">problem?</span>

In 1921, the U.S. government introduced both 8.5″ × 11″ and 8″ × 10.5″ as standard paper sizes. The smaller size caught on within U.S. government agencies; the larger, with everyone else. It wasn’t until 1980 that the whole U.S. government finally settled on <span class="no-hyphens">8.5″ × 11″.</span>

The 8.5″ × 11″ paper size, and the sizes of computer screens, are standards imposed on us by history and tradition. They are arbitrary. They do not represent anyone’s idea of a convenient size for good typography. But with page margins, you can reshape <span class="no-hyphens">them.</span>

As proof, consider a print example. Are there any publications that use 8.5″ × 11″ paper? Yes, it’s the approximate size of many magazines. But do any of those magazines run text in a single block on the page with one-inch margins? No—never. They use multiple-column layouts or find other ways to divide the <span class="no-hyphens">page.</span>

So are there any publications that *do* run text in a single block on the page? Sure—books are usually set in a single column. But do you ever see a book printed on 8.5″ × 11″ paper? No—never. It would be too big for comfortable <span class="no-hyphens">reading.</span>

Professional typographers never use 8.5″ × 11″ paper with a 6.5″ line length. Neither should you. Whether you’re writing for print or the web, set your text according to the principles of good typography. The white space will take care of itself. The pleasure of reading an effectively designed document will soon outweigh the unfamiliarity of extra white space around the <span class="no-hyphens">edges.</span>

“But with those big margins, I won’t get nearly as many words on the page.” Oh really? Let’s address that fear with a word-processor exercise that brings together some of what you’ve learned so <span class="no-hyphens">far.</span>

1.  Start a new document in your word processor. Paste in a text of at least 1000 <span class="no-hyphens">words.</span>

2.  With a 12-point font, “Double” line spacing in Word works out to be 28 points. (If that seems like peculiar math, I agree. See <a href="line-spacing.html" class="xref">line spacing</a>.) I’m using it here because it’s the line-spacing option many people select by <span class="no-hyphens">default.</span>

    Format this new document as follows: page margins of 1″ per side, font is <a href="times-new-roman-alternatives.html" class="xref">Times New Roman</a>, <a href="point-size.html" class="xref">point size</a> is 12, <a href="line-spacing.html" class="xref">line spacing</a> is “Double” (if you’re using Word; if not, use exactly 28 points), <a href="first-line-indents.html" class="xref">first-line indent</a> is 0.5″, and no space between paragraphs. I’ll call this document <span class="no-hyphens">A.</span>

3.  Start another new document in your word processor. Paste in the same <span class="no-hyphens">text.</span>

4.  Format this second document as follows: page margins of 2″ per side, font is still Times New Roman, point size is 11, line spacing is exactly 15 points, first-line indent is still 0.5″, and still no space between paragraphs. I’ll call this document <span class="no-hyphens">B.</span>

5.  Shortcut: the “before” and “after” samples in <a href="research-papers.html" class="xref">research papers</a> are pretty close to the formatting of documents A and B. <span class="no-hyphens"></span>

    Print both documents. Which one looks more like a professionally typeset book: A or <span class="no-hyphens">B?</span>

6.  Which document is more comfortable to read: A or <span class="no-hyphens">B?</span>

7.  Which document contains more words per page: A or B? Hint: use word count. See <a href="line-length.html" class="xref">line length</a> for <span class="no-hyphens">instructions.</span>

I’m guessing you answered B to the last three questions. If so, you’re seeing how good typography can be a benevolent force—it improves the appearance and legibility of your text with no compromise in words per <span class="no-hyphens">page.</span>

<div class="btw">

<div class="btw-title">

by the way

</div>

- A *gutter* margin is extra space on one side of a printed page that accounts for a binding. In a duplex (two-sided) document, the gutter will automatically alternate <span class="no-hyphens">sides.</span>

- In a printed document, do your margins all have to be the same size? No. To fit more text on the page, reduce the top and bottom margins. Your line length will stay the same, but you’ll get more lines per page. To make the text block appear centered vertically, try making the bottom margin about a 0.25″ larger than the top margin. Otherwise, the text block can look like it’s sagging. Finally, there’s no rule that a text block has to be centered on the page horizontally. For an asymmetric layout, make the difference between the left and right margins at least 1″ to make the asymmetry <span class="no-hyphens">obvious.</span>

- The best way to judge a layout is with your eyes, not with a calculator. But typographers have long enjoyed fiddling with layouts that incorporate specific mathematical ratios. Most famous among these is the golden ratio, which is approximately 1.618 : 1. For instance, if you set page margins of 2.23″ on all four sides of 8.5″ × 11″ paper, the proportions of your text block will be close to the golden <span class="no-hyphens">ratio.</span>
