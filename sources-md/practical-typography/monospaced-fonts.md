monospaced fontsDon’t use these either

The <a href="system-fonts.html" class="xref">system fonts</a> Courier, Menlo, and Consolas are examples of monospaced fonts, so named because every character is the same width. When the characters vary in width, the font is called proportional.

|  |
|----|
| abcdefghijklmnopqrstuvwxyz! |
| <span style="font-family:valkyrie-text">abcdefghijklmnopqrstuvwxyz!</span> |
| Jill, did you buy the milk? |
| <span style="font-family:valkyrie-text">Jill, did you buy the milk?</span> |

The samples above are set at the same <a href="point-size.html" class="xref">point size</a>. But the monospaced font, (first and third rows) takes up more horizontal space than the proportional font, (second and fourth rows). The differences are most noticeable in characters that are narrow in the proportional font (like f i j l r t) and the punctuation <span class="no-hyphens">characters.</span>

In 1944, IBM introduced a typewriter that could handle proportional fonts. But the technology never took <span class="no-hyphens">off.</span>

Monospaced fonts were invented to suit the mechanical requirements of typewriters. They were not invented to win beauty contests. Compared to proportional fonts, monospaced fonts are harder to read. And because they take up more horizontal space, you’ll always get fewer words per page with a monospaced <span class="no-hyphens">font.</span>

In standard <a href="body-text.html" class="xref">body text</a>, there are no good reasons to use monospaced fonts. So don’t. Use proportional <span class="no-hyphens">fonts.</span>

<div class="btw">

<div class="btw-title">

by the way

</div>

- Do you need monospaced numerals—typographers call them tabular figures—so that columns of numbers will line up? This is such a common need that most proportional fonts include tabular figures by default. See <a href="alternate-figures.html" class="xref">alternate figures</a> for how to verify <span class="no-hyphens">this.</span>

- Do you need to quote software code or HTML in your document? Then use a monospaced font for one of the reasons software engineers do—software code includes compressed syntax like (int i=1; i\<111; i++) which is more legible when set in a monospaced font `(int i=1; i<111; i++)`.

- If you really do need a monospaced font, the Courier system font is one of the worst. See <a href="courier-alternatives.html" class="xref">Courier alternatives</a>.
