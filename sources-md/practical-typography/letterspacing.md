letterspacingUse 5–12% extra space with caps, but not with lowercase

Letterspacing (also known as character spacing or tracking) is the adjustment of the horizontal white space between the letters in a block of text. Unlike <a href="kerning.html" class="xref">kerning</a>, which affects only designated pairs of letters, letterspacing affects every <span class="no-hyphens">pair.</span>

<div class="indented">

</div>

Lowercase letters don’t ordinarily need <span class="no-hyphens">letterspacing.</span>

Capital letters usually appear at the beginning of a word or sentence, so they’re designed to fit correctly next to lowercase letters. But when you use capital letters together, that spacing looks too tight. That’s why you always add 5–12% extra letterspacing to text in <a href="all-caps.html" class="xref">all caps</a> or <a href="small-caps.html" class="xref">small caps</a>. This is particularly important at small sizes.

<div class="howto">

<div class="howto-name">

How to set letterspacing

</div>

WordRight-click in the text and select Font from the menu. Click the Advanced tab. On the line that says Spacing, in the box on the right, enter the amount of letterspacing. Letterspacing in Word is measured in points. Use 0.6–1.4 points of letterspacing for every 12 points of point size (this corresponds to <span class="no-hyphens">5–12%).</span>

PagesView → Show Toolbar (or option + ⌘ + t) → Format button → Style pane → under the Font section, click the gear icon and use the Character Spacing box to add <span class="no-hyphens">5–12%.</span>

CSSSet the `letter-spacing` property in the range `0.05em` to `0.12em` (an `em` is 100% of the font size, so this is equivalent to <span class="no-hyphens">5–12%).</span>

</div>

Architects love to do this with lettering on building exteriors. I wish they’d <span class="no-hyphens">stop.</span>

These are not absolute limits—use your judgment. But avoid the common error of spreading letters too far apart. If the spaces between letters are large enough to fit more letters, you’ve gone <span class="no-hyphens">overboard.</span>

<div class="btw">

<div class="btw-title">

by the way

</div>

- blackletter <span class="no-hyphens">font</span>

  Typographer Frederic Goudy is famously credited with opining that “Anyone who would letterspace lowercase would steal sheep”. But a few sources claim that his original comment concerned blackletter fonts, not lowercase, and that he used a more colorful verb than <span class="no-hyphens">“steal”.</span>

- I accept the minority view on Goudy’s comment because, as Goudy was doubtless aware, sometimes lowercase should be letterspaced. Fonts intended for <a href="body-text.html" class="xref">body text</a> have spacing optimized for body-text point sizes (approximately 9–13 point). But typographers will often add letterspacing to lowercase text smaller than 9 point in order to keep the spaces between letters distinct. Similarly, typographers will often remove letterspacing from lowercase text used at larger sizes (e.g., <span class="no-hyphens">headlines).</span>

- As with <a href="kerning.html" class="xref">kerning</a>, if you use <a href="paragraph-and-character-styles.html" class="xref">paragraph and character styles</a> to make a style with all caps or small caps, include letterspacing as part of the style <span class="no-hyphens">definition.</span>
