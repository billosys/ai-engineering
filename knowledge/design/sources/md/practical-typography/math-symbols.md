math symbolsUse real symbols, not alphabetic characters

If you need math symbols infrequently—some light addition and subtraction, the occasional negative number—you can get by without special symbols. Use the plus sign (+) and equals sign (=) as usual. Use the en dash as a minus sign (–) (see <a href="hyphens-and-dashes.html" class="xref">hyphens and dashes</a>).

If you need to do multiplication or beyond, insert real math symbols by <span class="no-hyphens">hand.</span>

|                    |       |
|--------------------|-------|
| 12 x 34 - 56 = 352 | wrong |
| 12 × 34 − 56 = 352 | right |

<div class="howto">

<div class="howto-name">

How to insert math symbols

</div>

WordInsert → Symbols panel → Symbol → More <span class="no-hyphens">Symbols</span>

Mac OS WordInsert menu → Advanced <span class="no-hyphens">Symbol</span>

PagesEdit → Emoji & Symbols (or control + ⌘ + space). In the resulting character palette, if you don’t see a list of symbol categories on the left, change the view by scrolling to the top of the palette and clicking the button in the upper right. From the list on the left, select Math Symbols. Double-click the symbol you want. (Special handling is needed for <a href="foot-and-inch-marks.html" class="xref">foot and inch marks</a>.)

HTMLMinus sign is `&minus;` multiplication sign is `&times;` division sign is `&divide;` ([many others](http://www.math.uh.edu/~hjm/HTML%20Tag%20List.htm) are also <span class="no-hyphens">supported)</span>

</div>

<div class="btw">

<div class="btw-title">

by the way

</div>

- The multiplication symbol is also properly part of dimensional <span class="no-hyphens">notations.</span>

  |            |       |
  |------------|-------|
  | 8.5″ x 14″ | wrong |
  | 8.5″ × 14″ | right |
