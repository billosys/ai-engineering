It’s counterintuitive, but a well-designed font can have a lot of subtle variation between letters and still look consistent on the page. The converse of this principle is that a font with too much consistency can be numbing to read. Cambria is an example of this problem. It works well on screen, but on the printed page, it’s a skull-clutcher. <a href="fontrec/guardian-egyptian.html" class="xref">Guardian Egyptian</a>, <a href="fontrec/elena.html" class="xref">Elena</a>, and <a href="fontrec/skolar.html" class="xref">Skolar</a> are similar to Cambria but avoid monotony. So are <a href="charter.html" class="xref">Charter</a> and <a href="fontrec/source-serif.html" class="xref">Source Serif</a>, which are available for <span class="no-hyphens">free.</span>
