Hermes Maia was inspired by German display type of the <span class="no-hyphens">1920s.</span>

Hermes Maia comes in six weights, with italics and small caps for every weight. It includes oldstyle and tabular <a href="alternate-figures.html" class="xref">alternate figures</a>.

Hermes Maia comes with a separate set of caps fonts, which contain real <a href="small-caps.html" class="xref">small caps</a> and already include my recommended <a href="letterspacing.html" class="xref">letterspacing</a>.

Build your own font bundle: your Hermes Maia order will include a discount code to get additional families for \$79 instead of \$119 (1–2 person license only). You can also upgrade anytime to the full MB Type Library for the difference in <span class="no-hyphens">price.</span>

Hermes Maia has a short, plain-English [font license](https://mbtype.com/license) that includes permission to use it in PDFs, websites, apps, and electronic books. It is a one-time payment, not a subscription.

Each license covers multiple people, but only within a single organization or family. A license cannot be split among friends or random strangers. [Details here.](https://mbtype.com/license)

Hermes Maia has no demo version, but I offer a 30-day return option: if the fonts aren’t your style, you can cancel your license for a <span class="no-hyphens">refund.</span>
