apostrophesMake sure they’re curly and point downward

The *apostrophe* has two functions we all remember from sixth-grade English <span class="no-hyphens">class.</span>

1.  An apostrophe indicates the possessive case *(Jessica’s <span class="no-hyphens">bagel).</span>*

2.  In contractions, an apostrophe takes the place of letters or numbers that have been removed (*is not* becomes *isn’t*, *Patent No. 5,269,211* becomes *’211*).

Apostrophes always point downward. If the smart-quote feature of your writing system is activated (see <a href="straight-and-curly-quotes.html" class="xref">straight and curly quotes</a>), you type an apostrophe with the same key you use to type a straight single quote ('). Your word processor will convert this character to a curly apostrophe (’). Or you can type an apostrophe directly, using the same key as a closing single <span class="no-hyphens">quote.</span>

|  |  | Windows | Mac OS | HTML |
|----|----|----|----|----|
| ' | straight single quote | ' | ' | ' |
| ’ | apostrophe (same as closing single quote) | alt 0146 | option + shift + \] | `&rsquo;` |

Text imported from a plain-text source (e.g., a web page or email ) may not have its apostrophes converted to curly apostrophes. To fix this, use the search-and-replace technique in <a href="straight-and-curly-quotes.html" class="xref">straight and curly quotes</a>.

Wrinkles arise when an apostrophe is used at the beginning of a word (again, assuming your smart-quote conversion is on). If you type the <span class="no-hyphens">phrase:</span>

|                            |
|----------------------------|
| In the '70s, rock 'n' roll |

This will be displayed <span class="no-hyphens">as:</span>

|                            |       |
|----------------------------|-------|
| In the ‘70s, rock ‘n’ roll | wrong |

No, you don’t need an apostrophe before the *s* in *’70s*.

The problem here is that the characters in front of *70s* and *n’* aren’t apostrophes—they’re opening single quotes. They point upward. What you need is an apostrophe in place of each sequence of omitted letters, so the result looks like this:

|                            |       |
|----------------------------|-------|
| In the ’70s, rock ’n’ roll | right |

Microsoft Word has its own apostrophe shortcut—hold down the control key and type the straight single quote <span class="no-hyphens">twice.</span>

To get this result, you have two choices. You can manually delete the incorrect marks and type apostrophes directly (using the key combinations above). Or you can type two single <span class="no-hyphens">quotes:</span>

|                              |
|------------------------------|
| In the ''70s, rock ''n' roll |

These will be displayed <span class="no-hyphens">as:</span>

|                              |
|------------------------------|
| In the ‘’70s, rock ‘’n’ roll |

Then you can delete the unneeded opening single <span class="no-hyphens">quotes:</span>

|                            |       |
|----------------------------|-------|
| In the ’70s, rock ’n’ roll | right |

<div class="btw">

<div class="btw-title">

by the way

</div>

- If you’re using Hawaiian spellings of Hawaiian words, look out. Those apostrophe-like characters aren’t apostrophes—they’re okinas. The okina is a letter in the Hawaiian alphabet that doesn’t exist in English. Okinas point upward, so use an opening single quote as your okina, not an apostrophe. If the okina appears in the middle of a word, your word processor will incorrectly insert an <span class="no-hyphens">apostrophe.</span>

  |                 |       |
  |-----------------|-------|
  | Hawai’i O’ahu’s | wrong |
  | Hawai‘i O‘ahu’s | right |

  Alternatively, you can omit the okinas. Anglicized spellings of Hawaiian words are almost always <span class="no-hyphens">acceptable.</span>

  |               |       |
  |---------------|-------|
  | Hawaii Oahu’s | right |

  The okina is a *glottal stop*. In English, the glottal stop is heard before some vowels, like in the middle of the word *uh-oh*.
