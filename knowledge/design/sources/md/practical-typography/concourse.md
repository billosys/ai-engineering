Concourse is one of the fonts available for <a href="body-text.html" class="xref">body text</a> in this book. (Click on the font-selection toolbar at the bottom of the page to try it.) It’s also used in this book for sidebars, technical instructions, and <span class="no-hyphens">labels.</span>

Concourse takes inspiration from the geometric sans serifs of the 1930s, but doesn’t skimp on versatility, warmth, or <span class="no-hyphens">personality.</span>

Concourse comes in six weights, with real italics and small caps for every weight. It includes oldstyle and tabular <a href="alternate-figures.html" class="xref">alternate figures</a>.

Concourse comes with a separate set of caps fonts, which contain real <a href="small-caps.html" class="xref">small caps</a> and already include my recommended <a href="letterspacing.html" class="xref">letterspacing</a>.

Build your own font bundle: your Concourse order will include a discount code to get additional families for \$79 instead of \$119 (1–2 person license only). You can also upgrade anytime to the full MB Type Library for the difference in <span class="no-hyphens">price.</span>

Concourse has a short, plain-English [font license](https://mbtype.com/license) that includes permission to use it in PDFs, websites, apps, and electronic books. It is a one-time payment, not a subscription.

Each license covers multiple people, but only within a single organization or family. A license cannot be split among friends or random strangers. [Details here.](https://mbtype.com/license)

Concourse has no demo version, but I offer a 30-day return option: if the fonts aren’t your style, you can cancel your license for a <span class="no-hyphens">refund.</span>
