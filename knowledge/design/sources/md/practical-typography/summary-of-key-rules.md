Summary of key rules

1.  The four most important typographic considerations for body text are <a href="point-size.html" class="xref">point size</a>, <a href="line-spacing.html" class="xref">line spacing</a>, <a href="line-length.html" class="xref">line length</a>, and font (see <a href="font-recommendations.html" class="xref">font recommendations</a>), because those choices determine how the <a href="body-text.html" class="xref">body text</a> looks.

2.  <a href="point-size.html" class="xref">Point size</a> should be 10–12 points in printed documents, 15-25 pixels on the web.

3.  <a href="line-spacing.html" class="xref">Line spacing</a> should be 120–145% of the point <span class="no-hyphens">size.</span>

4.  The average <a href="line-length.html" class="xref">line length</a> should be 45–90 characters (including spaces).

5.  The easiest and most visible improvement you can make to your typography is to use a professional font, like those found in <a href="font-recommendations.html" class="xref">font recommendations</a>.

6.  Avoid <a href="goofy-fonts.html" class="xref">goofy fonts</a>, <a href="monospaced-fonts.html" class="xref">monospaced fonts</a>, most <a href="free-fonts.html" class="xref">free fonts</a>, and <a href="system-fonts.html" class="xref">system fonts</a>—especially <a href="times-new-roman-alternatives.html" class="xref">Times New Roman</a> and <a href="helvetica-and-arial-alternatives.html" class="xref">Arial</a>.

7.  Use curly quotation marks, not straight ones (see <a href="straight-and-curly-quotes.html" class="xref">straight and curly quotes</a>).

8.  Use <a href="bold-or-italic.html" class="xref">bold or italic</a> as little as possible, and not <span class="no-hyphens">together.</span>

9.  Never <a href="underlining.html" class="xref">underline</a>, except perhaps for web links.

10. <a href="all-caps.html" class="xref">All caps</a> are fine for less than one line of <span class="no-hyphens">text.</span>

11. Use <a href="centered-text.html" class="xref">centered text</a> <span class="no-hyphens">sparingly.</span>

12. Put only <a href="one-space-between-sentences.html" class="xref">one space between sentences</a>.

13. Don’t use multiple <a href="word-spaces.html" class="xref">word spaces</a> or other <a href="white-space-characters.html" class="xref">white-space characters</a> in a <span class="no-hyphens">row.</span>

14. If you don’t have real <a href="small-caps.html" class="xref">small caps</a>, don’t use them at <span class="no-hyphens">all.</span>

15. Use 5–12% extra <a href="letterspacing.html" class="xref">letterspacing</a> with <a href="all-caps.html" class="xref">all caps</a> and <a href="small-caps.html" class="xref">small caps</a>.

16. <a href="kerning.html" class="xref">Kerning</a> should always be turned on.

17. Use <a href="first-line-indents.html" class="xref">first-line indents</a> that are one to four times the <a href="point-size.html" class="xref">point size</a> of the text, or use 4–10 points of <a href="space-between-paragraphs.html" class="xref">space between paragraphs</a>. Don’t use <span class="no-hyphens">both.</span>

18. Always use <a href="hyphenation.html" class="xref">hyphenation</a> with <a href="justified-text.html" class="xref">justified text</a>.

19. Don’t confuse <a href="hyphens-and-dashes.html" class="xref">hyphens and dashes</a>, and don’t use multiple hyphens as a <span class="no-hyphens">dash.</span>

20. Use <a href="ampersands.html" class="xref">ampersands</a> sparingly, unless included in a proper <span class="no-hyphens">name.</span>

21. Use proper <a href="trademark-and-copyright-symbols.html" class="xref">trademark and copyright symbols</a>—not alphabetic <span class="no-hyphens">approximations.</span>

22. In a document longer than three pages, one exclamation point is plenty (see <a href="question-marks-and-exclamation-points.html" class="xref">question marks and exclamation points</a>).

23. Put a <a href="nonbreaking-spaces.html" class="xref">nonbreaking space</a> after <a href="paragraph-and-section-marks.html" class="xref">paragraph and section marks</a>.

24. Make <a href="ellipses.html" class="xref">ellipses</a> using the proper character, not periods and <span class="no-hyphens">spaces.</span>

25. <a href="apostrophes.html" class="xref">Apostrophes</a> point <span class="no-hyphens">downward.</span>

26. Make sure <a href="foot-and-inch-marks.html" class="xref">foot and inch marks</a> are straight, not <span class="no-hyphens">curly.</span>
