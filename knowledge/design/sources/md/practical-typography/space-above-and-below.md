space above & belowElegant emphasis

If you hadn’t figured it out yet, I’m a big fan of white space. On the outside edge of the page (see <a href="page-margins.html" class="xref">page margins</a>). Between capital letters (see <a href="letterspacing.html" class="xref">letterspacing</a>). Within table cells (see <a href="tables.html" class="xref">tables</a>). Between paragraphs (see <a href="space-between-paragraphs.html" class="xref">space between paragraphs</a>).

And now, above and below a <a href="headings.html" class="xref">heading</a>, as a form of emphasis. To show you what I mean, let me ask you something. <span style="border-bottom:0.15em solid black">***DO YOU FIND THIS TO BE AN INVITING FORM OF <span class="no-hyphens">EMPHASIS?</span>***</span>

<div class="gap" style="height:7em">

</div>

Or do you prefer <span class="no-hyphens">this?</span>

<div class="gap" style="height:7em">

</div>

Well, there you have <span class="no-hyphens">it.</span>

Space above and below works on the page for the same reason a dramatic pause works when you’re talking. You draw a listener’s attention through contrast. Loudness is an effective form of contrast. But so is <span class="no-hyphens">silence.</span>

Likewise with the written word. Formatting tools like <a href="point-size.html" class="xref">point size</a>, <a href="color.html" class="xref">color</a>, and <a href="bold-or-italic.html" class="xref">bold or italic</a> have a role to play. But careless typographers mate them indiscriminately, breeding noise. Space above and below gives you emphasis without adding a single mark to the <span class="no-hyphens">page.</span>

<div class="howto">

<div class="howto-name">

How to set space above and below

</div>

WordRight-click in the text and select Paragraph → Indents and Spacing. Under Spacing, in the boxes next to Before and After, enter the <span class="no-hyphens">measurements.</span>

PagesView → Show Toolbar (or option + ⌘ + t) → Format button → Style pane → under Spacing, in the boxes labeled Before Paragraph and After Paragraph, enter the <span class="no-hyphens">measurements.</span>

CSSUse the `margin-top` and `margin-bottom` <span class="no-hyphens">properties</span>

</div>

How much space to add is a judgment call. Semantically, headings relate to the text that follows, not the text before. Thus you’ll probably want the space below to be smaller than the space above so the heading is visually closer to the text it <span class="no-hyphens">introduces.</span>

There is an upper limit. For instance, on a website, you wouldn’t want to add so much space before a heading that it disappears below the bottom of the <span class="no-hyphens">browser.</span>

<div class="btw">

<div class="btw-title">

by the way

</div>

- If you’re already using space between paragraphs, the space you add around a heading should be larger, to create a <span class="no-hyphens">distinction.</span>

</div>
