# Competency Questions for Cargo Getting Started

## Definitional (What is X?)
1. What is rustup?
2. What is a Cargo package?
3. What is a crate in the Cargo ecosystem?
4. What is a binary crate?
5. What is a library crate?
6. What is a Cargo manifest (Cargo.toml)?
7. What are Rust release channels?

## Relational (How does X relate to Y?)
8. How does rustup relate to Cargo installation?
9. How does a package relate to a crate?
10. How does Cargo.toml relate to a package?

## Procedural (How do I do X?)
11. How do I install Rust and Cargo?
12. How do I create a new Cargo package?
13. How do I build a Cargo project?
14. How do I compile and run a Cargo project in one step?

## Prerequisite (What before X?)
15. What do I need before I can use Cargo?

## Diagnostic (What distinguishes X from Y?)
16. What is the difference between a binary crate and a library crate?
17. What is the difference between `cargo build` and `cargo run`?
18. What is the difference between stable, beta, and nightly release channels?
