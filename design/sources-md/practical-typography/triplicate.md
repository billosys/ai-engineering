Triplicate is used for code samples and other monospaced typesetting in this book (e.g., <a href="one-space-between-sentences.html" class="xref">one space between sentences</a>). It’s also one of the fonts available for <a href="body-text.html" class="xref">body text</a> in this book. (Click on the font-selection toolbar at the bottom of the page to try <span class="no-hyphens">it.)</span>

True, my official advice is that in general you should avoid <a href="monospaced-fonts.html" class="xref">monospaced fonts</a>. But in some cases, a monospaced font is the right tool for the job. So why not have a nice <span class="no-hyphens">one?</span>

Triplicate has true italics, real small caps, oldstyle figures, swash caps, box-drawing characters, and other <span class="no-hyphens">superfluities.</span>

Triplicate includes a Code variant with characters optimized for programming and terminal windows (also seen in <a href="why-racket-why-lisp.html" class="xref">Why Racket? Why Lisp?</a>).
