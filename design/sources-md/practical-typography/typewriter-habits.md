Typewriter habits

I’ve claimed throughout this book that many bad typography habits have been left over from the typewriter era. Here, I’ve collected them in one list. (Reminder—these are things you should **not** <span class="no-hyphens">do.)</span>

1.  Straight quotes rather than curly quotes (see <a href="straight-and-curly-quotes.html" class="xref">straight and curly quotes</a>).

2.  Two spaces rather than <a href="one-space-between-sentences.html" class="xref">one space between sentences</a>.

3.  Multiple hyphens instead of dashes (see <a href="hyphens-and-dashes.html" class="xref">hyphens and dashes</a>).

4.  Alphabetic approximations of <a href="trademark-and-copyright-symbols.html" class="xref">trademark and copyright symbols</a>.

5.  <a href="ellipses.html" class="xref">ellipses</a> made with three periods rather than an ellipsis <span class="no-hyphens">character.</span>

6.  Non-curly <a href="apostrophes.html" class="xref">apostrophes</a>.

7.  Pretending that <a href="accented-characters.html" class="xref">accented characters</a> don’t <span class="no-hyphens">exist.</span>

8.  Using more than one <a href="word-spaces.html" class="xref">word space</a> at a <span class="no-hyphens">time.</span>

9.  Using <a href="tabs-and-tab-stops.html" class="xref">tabs and tab stops</a> instead of <a href="tables.html" class="xref">tables</a>.

10. Using <a href="carriage-returns.html" class="xref">carriage returns</a> to insert vertical <span class="no-hyphens">space.</span>

11. Using alphabetic characters as substitutes for real <a href="math-symbols.html" class="xref">math symbols</a>.

12. Making <a href="rules-and-borders.html" class="xref">rules and borders</a> out of repeated alphabetic <span class="no-hyphens">characters.</span>

13. Ignoring <a href="ligatures.html" class="xref">ligatures</a>.

14. <a href="underlining.html" class="xref">Underlining</a> <span class="no-hyphens">anything.</span>

15. Using <a href="monospaced-fonts.html" class="xref">monospaced fonts</a> rather than proportional <span class="no-hyphens">fonts.</span>

16. Abusing <a href="all-caps.html" class="xref">all caps</a>.

17. Thinking that the best <a href="point-size.html" class="xref">point size</a> for body text is <span class="no-hyphens">12.</span>

18. Ignoring <a href="kerning.html" class="xref">kerning</a>.

19. Ignoring <a href="letterspacing.html" class="xref">letterspacing</a>.

20. Too much <a href="centered-text.html" class="xref">centered text</a>.

21. Only using single or double <a href="line-spacing.html" class="xref">line spacing</a>.

22. Only using the <a href="line-length.html" class="xref">line length</a> permitted by one-inch <a href="page-margins.html" class="xref">page margins</a>.

<div class="btw-title">

by the way

</div>

- I’m far from the first person to observe that many bad typographic habits have been passed down from typewriters. A bestselling title of the early Mac era was Robin Williams’s book *The Mac is Not a Typewriter*, which spawned a sequel, *The PC is Not a Typewriter*. Still <span class="no-hyphens">true.</span>
