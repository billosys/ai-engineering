How to use this book

The bottom of each page has navigational links for the previous page, next page, top page, and chapter index (where applicable). You can change the <a href="body-text.html" class="xref">body text</a> font by clicking on a font <span class="no-hyphens">name.</span>

Furthermore, you <span class="no-hyphens">can:</span>

- click the **left edge** for the previous <span class="no-hyphens">page.</span>

- click the **right edge** for the next <span class="no-hyphens">page.</span>

Roll over the page edges to see these clickable <span class="no-hyphens">areas.</span>

Use the font picker at the bottom of the page to change the font used for <a href="body-text.html" class="xref">body text</a>. The font picker can be detached and moved around the <span class="no-hyphens">page.</span>

Cross-references, denoted with <a href="small-caps.html" class="xref">small caps</a>, are clickable. This is an homage to reference works like *Garner’s Modern English Usage* (see <a href="bibliography.html" class="xref">bibliography</a>), which commonly format cross-references in the same style. If you own one of these books, you’re familiar. If you don’t, buy <span class="no-hyphens">one.</span>

Links to outside material are denoted with a red circle, [like so.](#) For three <span class="no-hyphens">reasons:</span>

1.  To distinguish them visually from internal cross-references—a small signal that “hey, you’re now leaving the book”. (Occasionally I bend this rule, but don’t <span class="no-hyphens">panic.)</span>

2.  Yes, I’m aware that web hyperlinks have typically been underlined. But <a href="underlining.html" class="xref">underlining</a> is nothing more than an odious <a href="typewriter-habits.html" class="xref">typewriter habit</a>, held over on <a href="websites.html" class="xref">websites</a> because of inertia. It was tolerable in 1994 when web browsers could do so little. But not today, when they can do so <span class="no-hyphens">much.</span>

3.  Vigorously styled hyperlinks on a page tend to move to the foreground of a reader’s attention, like an HDTV in a hotel bar. (See also <a href="maxims-of-page-layout.html" class="xref">maxims of page layout</a>.) The red circle is meant to be noticeable while you’re reading the sentence that contains the link. Otherwise it disappears, so as not to <span class="no-hyphens">distract.</span>

Of course, the main reason hyperlinks have historically been overstyled is because generating clicks on those links is essential to internet advertising. Here, you will find no advertising—[just us readers.](how-to-pay-for-this-book.md) The design is tailored accordingly. <span class="no-hyphens"></span>
