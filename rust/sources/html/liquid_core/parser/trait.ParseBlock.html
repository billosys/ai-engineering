Expand description

A trait for creating custom custom block-size tags ({% if something %}{% endif %}). This is a simple type alias for a function.

This function will be called whenever the parser encounters a block and returns a new Renderable based on its parameters. The received parameters specify the name of the block, the argument Tokens passed to the block, a TagBlock inside the block and the global Language.
